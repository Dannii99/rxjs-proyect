Observable

Una representación de cualquier conjunto de valores durante cualquier período de tiempo. Este es el bloque de construcción más básico de RxJS.

@for (item of obs; track $index; let i = $index) {

value({{i+1}}): {{item}}

}

Observer

Una interfaz de objeto que define un conjunto de funciones de devolución de llamada que un usuario puede usar para obtener notificado de cualquier conjunto de eventos de notificación observables.

}

Subject

Un Subject es un tipo especial de Observable que permite que los valores sean multidifusión a muchos observadores. Los Subjects son como EventEmitters.

BehaviorSubject

Una variante de Subject que requiere un valor inicial y emite su valor cada vez que se suscribe.

@for (item of behaviorSubject; track $index; let i = $index) {

value({{i+1}}): {{item}}

}

ReplaySubject

Una variante de Subject que "reproduce" los valores antiguos a los nuevos suscriptores emitiéndolos cuando se suscriben por primera vez.
